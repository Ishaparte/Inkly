# Book-Ecommerce — Frontend

Lightweight, component-driven frontend for a book e-commerce application. Built with React  to sell and buy books, manage a cart, authenticate users, and complete purchases.

## Feature
- Shopping cart & checkout flow
- User authentication (login / register)
- Order history
- Admin pages for managing books
- Responsive design

## Tech stack
- React 
- CSS 
- Vite 
- React Router for navigation
- Google pay API payment integration
- Firebase Auth
